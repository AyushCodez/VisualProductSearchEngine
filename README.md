# Visual Product Search — Streamlit App

Interactive end-to-end pipeline for the VR Final Project. 

**Pipeline:**
1. **Upload** an image.
2. **Detect & Crop** garments automatically using a fine-tuned YOLOv8 fashion model.
3. **Confirm** the crop or select a different bounding box.
4. **Visual Search** the chosen crop against a local catalog using a custom Multimodal Pipeline (CLIP Image & Text Embeddings + BLIP Captions + FAISS).
5. **View Results** and similarity scores.

---

## 🛠️ Setup & Installation

### 1. Virtual Environment (Recommended)
It is highly recommended to run this project in a Python virtual environment to prevent dependency conflicts.

```bash
# Create a virtual environment
python -m venv venv

# Activate the virtual environment
# On Windows:
venv\Scripts\activate
# On Mac/Linux:
source venv/bin/activate
```

### 2. Install Dependencies
Install all required machine learning and UI libraries:

```bash
pip install -r requirements.txt
```

### 3. Required Data & Model Files
Before running the app, ensure you have the following files located in your project directory:

- **`data/` folder containing:**
  - `best_fashion_clip.pth` (Fine-tuned CLIP weights)
  - `optimized_fusion_faiss.index` (FAISS database)
  - `gallery_paths.npy` & `gallery_item_ids.npy` (Product metadata)
- **`Img/` folder containing:**
  - The DeepFashion gallery images (e.g. `Img/img/WOMEN/...`).
- **`yolov8n_fashion.pt`**: 
  - Automatically downloaded to your root directory when the app starts, handling the clothing-specific object detection.

---

## 🚀 How to Run

Once your virtual environment is activated and dependencies are installed, simply launch the Streamlit server:

```bash
streamlit run app.py
```

The application will automatically open in your default web browser at `http://localhost:8501`.

---

## 🧠 Architecture Overview
- **Object Detection:** Powered by `ultralytics` YOLOv8. Replaced the generic COCO model with a fashion-specific model to accurately crop individual garments.
- **Multimodal Retrieval:** Uses `transformers` to blend visual features (`openai/clip-vit-base-patch32`) and semantic text descriptions generated by BLIP (`Salesforce/blip-image-captioning-large`).
- **Indexing:** Fast L2/Inner Product nearest-neighbor search powered by `faiss-cpu`.
